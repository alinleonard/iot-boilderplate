# VueExpress-boilerplate

Frontend
----------
- [Vue-cli](https://github.com/vuejs/vue-cli/)
- [Axios](https://github.com/axios/axios)

Backend
----------
- Mongodb
- Express
- [Cors](https://github.com/expressjs/cors)


Installation (Frontend)
----------
	$ cd frontend
	$ npm install


Installation (Backend)
----------
	$ cd backend
	$ npm install


Usage (Frontend)
----------
	$ cd frontend
	$ npm run dev


Usage (Backend)
----------
	$ cd backend
	$ npm start
